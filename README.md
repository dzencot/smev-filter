*Описание проекта*

## Требования

* Node.js >= 14
* npm >= 6
* make >= 4

## Установка и запуск

* `make install` установка зависимостей
* `make setup` установка (первый раз)
* `smev-filter -h` запуск утилиты (вызов справки)

## Дополнительно

* `make lint` проверка линтером
* `make test` проверка тестами
